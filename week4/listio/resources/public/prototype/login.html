Methodology

Project idea is based on Eisenhower matrix: prioritizing tasks by urgency and importance.

However in software development another factor is also important: Clarity.
So, we took clarity and importance, and implemented the matrix for your project github issues.

Project features:

  • sync issues with github repository via custom tags (both ways)
  • save issues order (temporary disabled for stability)
  • daily boards (to be implemented)
  • board sharing (to be implemented)
